This is an experimental project to build a client and analysis tool for voting data retrieved from the Norges Bank Investment Management Voting Records API. It builds on the Python client by [Ömer Faruk Demir](https://github.com/omerfarukdemir). 

The project is currently under development. It currently contains a FastAPI access point to the NBIM client, set up to run as a Docker container listening on port 8088. The goal is to develop a HTML/JS based interface and to retrieve data to a local MySQL database.

To build on the project in any way, you will need to [apply to NBIM](https://www.nbim.no/en/the-fund/responsible-investment/our-voting-records/api-access-to-our-voting/) for an API key.
